## Description

* プログラム制御の対戦相手（NPC）とトランプの神経衰弱ゲームで遊ぶことができます
* NPCはまるで人間のように表になったカードの配置を記憶したり忘れたりします
* BootstrapとjQueryを試すための個人的なプロジェクトです

## Environments

* CSS/Bootstrap
* JavaScript/jQuery
* Firebase Hosting
